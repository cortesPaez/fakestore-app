
## Getting Started

### Prerequisites

Install expo cli
* npm
  ```sh
  npm install -g expo-cli exp
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/cortesPaez/fakestore-app
   ```
2. Install NPM packages
   ```sh
   expo install
   ```
3. Start project and select your system Android or iOs with the options of Expo Go
   ```sh
   expo start
   ```
